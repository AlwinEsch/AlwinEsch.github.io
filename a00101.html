Detailed Description

Kodi's keyboard class.

xbmc.Keyboard([default, heading, hidden])

Creates a new Keyboard object with default text heading and hidden input flag if supplied.

Parameters
default: [opt] string - default text entry.
heading: [opt] string - keyboard heading.
hidden: [opt] boolean - True for hidden text entry.

Example:

1 ..
2 kb = xbmc.Keyboard('default', 'heading', True)
3 kb.setDefault('password') # optional
4 kb.setHeading('Enter password') # optional
5 kb.setHiddenInput(True) # optional
6 kb.doModal()
7 if (kb.isConfirmed()):
8  text = kb.getText()
9 ..
