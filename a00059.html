Detailed Description

A standard push button control.

ControlButton(x, y, width, height, label[, focusTexture, noFocusTexture, textOffsetX, textOffsetY, alignment, font, textColor, disabledColor, angle, shadowColor, focusedColor])

The button control is used for creating push buttons in Kodi. You can choose the position, size, and look of the button, as well as choosing what action(s) should be performed when pushed.

Note
This class include also all calls from Control
Parameters
[in]xinteger - x coordinate of control.
[in]yinteger - y coordinate of control.
[in]widthinteger - width of control.
[in]heightinteger - height of control.
[in]labelstring or unicode - text string.
[in]focusTexture[opt] string - filename for focus texture.
[in]noFocusTexture[opt] string - filename for no focus texture.
[in]textOffsetX[opt] integer - x offset of label.
[in]textOffsetY[opt] integer - y offset of label.
[in]alignment[opt] integer - alignment of label
  • Flags for alignment used as bits to have several together:
    Defination name Bitflag Description
    XBFONT_LEFT 0x00000000 Align X left
    XBFONT_RIGHT 0x00000001 Align X right
    XBFONT_CENTER_X 0x00000002 Align X center
    XBFONT_CENTER_Y 0x00000004 Align Y center
    XBFONT_TRUNCATED 0x00000008 Truncated text
    XBFONT_JUSTIFIED 0x00000010 Justify text
[in]font[opt] string - font used for label text. (e.g. 'font13')
[in]textColor[opt] hexstring - color of enabled button's label. (e.g. '0xFFFFFFFF')
[in]disabledColor[opt] hexstring - color of disabled button's label. (e.g. '0xFFFF3300')
[in]angle[opt] integer - angle of control. (+ rotates CCW, - rotates CW)
[in]shadowColor[opt] hexstring - color of button's label's shadow. (e.g. '0xFF000000')
[in]focusedColor[opt] hexstring - color of focused button's label. (e.g. '0xFF00FFFF')
Note
You can use the above as keywords for arguments and skip certain optional arguments.
Once you use a keyword, all following arguments require the keyword.
After you create the control, you need to add it to the window with addControl().

Example:

1 ...
2 # ControlButton(x, y, width, height, label[, focusTexture, noFocusTexture, textOffsetX, textOffsetY,
3 # alignment, font, textColor, disabledColor, angle, shadowColor, focusedColor])
4 self.button = xbmcgui.ControlButton(100, 250, 200, 50, 'Status', font='font14')
5 ...
