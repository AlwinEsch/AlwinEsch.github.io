static bool CreateDirectory ( const std::string &  strPath)
static

Make a directory.

The CAddonLib_Directory::CreateDirectory(...) function shall create a new directory with name path.

The newly created directory shall be an empty directory.

Parameters
[in]strPathPath to the directory.
Returns
Upon successful completion, CreateDirectory() shall return true. Otherwise,false shall be returned, no directory shall be created, and errno shall be set to indicate the error.

Example:

#include <kodi/addon.api2/Directory.h>
...
std::string directory = "C:\\my_dir";
bool ret = CAddonLib_Directory::CreateDirectory(directory);
fprintf(stderr, "Directory '%s' successfull created: %s\n", directory.c_str(), ret ? "yes" : "no");
...

Verifying the Existence of a Directory.

The CAddonLib_Directory::DirectoryExists(...) method determines whether a specified folder exists.

Removes a directory.

The CAddonLib_Directory::RemoveDirectory(...) function shall remove a directory whose name is given by path.

Parameters
[in]strPathPath to the directory.
Returns
Upon successful completion, the function RemoveDirectory() shall return true. Otherwise, false shall be returned, and errno set to indicate the error. If false is returned, the named directory shall not be changed.

Example:

#include <kodi/addon.api2/Directory.h>
...
bool ret = CAddonLib_Directory::RemoveDirectory("C:\\my_dir");
