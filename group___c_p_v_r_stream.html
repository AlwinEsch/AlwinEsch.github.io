CPVRStream
C++ Development » Library - PVRLIB

Detailed Description

PVR stream properties data class

Represents a single stream. It extends the PODS to provide some operators overloads.

It has the header #include <kodi/pvr.api2/StreamUtils.h> be included to enjoy it.
