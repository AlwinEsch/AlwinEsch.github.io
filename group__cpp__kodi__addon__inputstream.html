Detailed Description

Class: kodi::addon::CInstanceInputStream

Inputstream add-on instance

It has the header #include <kodi/inputstream/InputStream.h> be included to enjoy it.

Here as example what is minimum required to start on a audio decoder:

#include <kodi/inputstream/InputStream.h>
// Todo: Finish example
/*----------------------------------------------------------------------*/
class CMyAddon : public ::kodi::addon::CAddonBase
{
public:
CMyAddon() { }
ADDON_STATUS CreateInstance(int instanceType,
std::string instanceID,
KODI_HANDLE instance,
KODI_HANDLE& addonInstance) override;
};
/* If you use only one instance in your add-on, can be instanceType and
* instanceID ignored */
ADDON_STATUS CMyAddon::CreateInstance(int instanceType,
std::string instanceID,
KODI_HANDLE instance,
KODI_HANDLE& addonInstance)
{
kodi::Log(LOG_NOTICE, "Creating my Inputstream");
addonInstance = new CMyInputstream(instance);
return ADDON_STATUS_OK;
}
ADDONCREATOR(CMyAddon);

The desctruction of the here created example class CMyInputstream becomes done from Kodi's header and never for add-on instances a delete on add-on itself necessary.
