Detailed Description

Virtual file system directory entry

This class is used as an entry for files and folders in KodiAPI::AddOn::CVFSUtils::GetDirectory(...).

Example:

#include <kodi/api3/addon/VFSUtils.hpp>
std::vector<KodiAPI::AddOn::CVFSDirEntry> items;
KodiAPI::AddOn::CVFSUtils::GetDirectory("special://temp", "", items);
fprintf(stderr, "Directory have %lu entries\n", items.size());
for (unsigned long i = 0; i < items.size(); i++)
{
fprintf(stderr, " - %04lu -- Folder: %s -- Name: %s -- Path: %s\n",
i+1,
items[i].IsFolder() ? "yes" : "no ",
items[i].Label().c_str(),
items[i].Path().c_str());
}

It has the header #include <kodi/api3/addon/VFSUtils.hpp> be included to enjoy it.
