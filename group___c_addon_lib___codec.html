CAddonLib_Codec
C++ Development » Library - AddOnLIB

Detailed Description

Demux packet handle and codec identification class

This class brings the support for obtaining required codec identifications and packages for handling stream demuxer.

Currently, this class is only required under PVR systems where by CAddonLib_Codec::AllocateDemuxPacket(...) a package for the function DemuxPacket* DemuxRead (void); must be provided.

These are pure static functions them no other initialization need.

It has the header #include <kodi/addon.api2/Codec.h> to enjoy it.

typedef struct DemuxPacket
{

static kodi_codec GetCodecByName ( const std::string &  strCodecName)
static

Get the codec id used by Kodi.

This function is required on registration of streams to Kodi, what is currently needed on the PVR add-on system where the demuxing becomes handled from them.

About available codec/demuxer names see see ffmpeg allformats.c in function void av_register_all(void) with on macro REGISTER_DEMUXER defined values. Further are inside Kodi the two names TELETEXT and RDS used to get kodi_codec value.

Parameters

Returns
The codec_id, or a codec_id with 0 values when not supported
