Enumeration Type Documentation

enum ADDON_VideoInfoTag

Video info tag data transfer flags.

Flags are used to identify data send from Add-on to Kodi and is used by the following functions:

  • CListItem::SetVideoInfo_BOOL(...);
  • CListItem::SetVideoInfo_INT(...);
  • CListItem::SetVideoInfo_UINT(...);
  • CListItem::SetVideoInfo_FLOAT(...);
  • CListItem::SetVideoInfo_STRING(...);
  • CListItem::SetVideoInfo_STRING_LIST(...);
Note
The end string of value shows the type of function to use.
