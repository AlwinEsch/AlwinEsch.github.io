## *Text viewer dialog*

General

The text viewer dialog can be used to display descriptions, help texts or other larger texts.

In order to achieve a line break is a \n directly in the text or in the **"./resources/language/resource.language.??_??/strings.po"** to call with `std::string CAddOnLib_General::GetLocalizedString(...);`.

It has the header `#include <kodi/gui.api2/GUIDialogTextViewer.h>` be included to enjoy it.
