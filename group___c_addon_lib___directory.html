CAddonLib_Directory
C++ Development » Library - AddOnLIB

Detailed Description

Directory system Programming

The Directory module offers functions that allow you to control it.

These are pure static functions them no other initialization need.

It has the header #include <kodi/addon.api2/Directory.h> be included to enjoy it.

static bool CreateDirectory ( const std::string &  strPath)
static

Make a directory.

The CAddonLib_Directory::CreateDirectory(...) function shall create a new directory with name path.

The newly created directory shall be an empty directory.

Parameters
strPathPath to the directory.
Returns
Upon successful completion, CreateDirectory() shall return true. Otherwise,false shall be returned, no directory shall be created, and errno shall be set to indicate the error.

Example:

#include <kodi/addon.api2/Directory.h>
...
std::string directory = "C:\\my_dir";
bool ret = CAddonLib_Directory::CreateDirectory(directory);
fprintf(stderr, "Directory '%s' successfull created: %s\n", directory.c_str(), ret ? "yes" : "no");
...
