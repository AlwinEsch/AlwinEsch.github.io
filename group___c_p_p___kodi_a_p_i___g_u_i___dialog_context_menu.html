Dialog Context Menu

Detailed Description

Context menu dialog

The function listed below permits the call of a dialogue as context menu to select of an entry as a key

These are pure static functions them no other initialization need.

It has the header #include <kodi/api2/gui/DialogContextMenu.hpp> be included to enjoy it.

Function Documentation

int kodi::gui::DialogContextMenu::Show ( const std::string &  heading,
const std::vector< std::string > &  entries 
)

Show a context menu dialog about given parts.

Parameters
[in]headingDialog heading name
[in]entriesString list about entries
Returns
The selected entry, if return -1 was nothing selected or canceled

Example:

#include <kodi/api2/gui/DialogContextMenu.hpp>
const std::vector<std::string> entries
{
"Test 1",
"Test 2",
"Test 3",
"Test 4",
"Test 5"
};
int selected = KodiAPI::GUI::DialogContextMenu::Show("Test selection", entries);
if (selected < 0)
fprintf(stderr, "Item selection canceled\n");
else
fprintf(stderr, "Selected item is: %i\n", selected);
