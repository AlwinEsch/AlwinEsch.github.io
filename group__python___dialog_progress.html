create (   ...)

Function: xbmcgui.DialogProgress().create(heading[, line1, line2, line3])

Create and show a progress dialog.

Parameters
headingstring or unicode - dialog heading.
line1[opt] string or unicode - line #1 multi-line text.
line2[opt] string or unicode - line #2 text.
line3[opt] string or unicode - line #3 text.
Note
It is preferred to only use line1 as it is actually a multi-line text. In this case line2 and line3 must be omitted.
Use update() to update lines and progressbar.

Example:

1 ..
2 pDialog = xbmcgui.DialogProgress()
3 pDialog.create('Kodi', 'Initializing script...')
4 ..
update (   ...)

Function: xbmcgui.DialogProgress().update(percent[, line1, line2, line3])

Updates the progress dialog.

Parameters
percentinteger - percent complete. (0:100)
line1[opt] string or unicode - line #1 multi-line text.
line2[opt] string or unicode - line #2 text.
line3[opt] string or unicode - line #3 text.
Note
It is preferred to only use line1 as it is actually a multi-line text. In this case line2 and line3 must be omitted.
If percent == 0, the progressbar will be hidden.

Example:

1 ..
2 pDialog.update(25, 'Importing modules...')
3 ..
