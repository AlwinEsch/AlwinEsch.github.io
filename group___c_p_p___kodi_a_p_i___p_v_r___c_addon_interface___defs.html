struct PVR_ADDON_CAPABILITIES

PVR add-on capabilities. All capabilities are set to "false" as default.

If a capabilty is set to true, then the corresponding methods from the several types need to be implemented.

Class Members
bool bSupportsEPG true if the add-on provides EPG information
bool bSupportsTV true if this add-on provides TV channels
bool bSupportsRadio true if this add-on supports radio channels
bool bSupportsRecordings true if this add-on supports playback of recordings stored on the backend
bool bSupportsRecordingsUndelete true if this add-on supports undelete of recordings stored on the backend
bool bSupportsTimers true if this add-on supports the creation and editing of timers
bool bSupportsChannelGroups true if this add-on supports channel groups
bool bSupportsChannelScan

true if this add-on support scanning for new channels on the backend with function:

  • KodiAPI::PVR::CAddonInterface::OpenDialogChannelScan()
bool bSupportsChannelSettings

true if this add-on supports the following functions:

  • KodiAPI::PVR::CAddonInterface::DeleteChannel()
  • KodiAPI::PVR::CAddonInterface::RenameChannel()
  • KodiAPI::PVR::CAddonInterface::MoveChannel()
  • KodiAPI::PVR::CAddonInterface::OpenDialogChannelSettings()
  • KodiAPI::PVR::CAddonInterface::OpenDialogChannelAdd()
bool bHandlesInputStream

true if this add-on provides an input stream. false if Kodi handles the stream.

bool bHandlesDemuxing true if this add-on demultiplexes packets.
bool bSupportsRecordingPlayCount true if the backend supports play count for recordings.
bool bSupportsLastPlayedPosition

true if the backend supports store/retrieve of last played position for recordings.

bool bSupportsRecordingEdl

true if the backend supports retrieving an edit decision list for recordings.

struct PVR_MENUHOOK

Menu hooks that are available in the context menus while playing a stream via this add-on. And in the Live TV settings dialog.

This structure becomes used from KodiAPI::PVR::General::AddMenuHook() and KodiAPI::PVR::CAddonInterface::CallMenuHook().

Class Members
unsigned int iHookId (required) this hook's identifier
unsigned int iLocalizedStringId (required) the id of the label for this hook in g_localizeStrings
PVR_MENUHOOK_CAT category (required) category of menu hook
struct PVR_MENUHOOK_DATA

PVR menu hook data.

Class Members
PVR_MENUHOOK_CAT cat category used for menu hook data
union data data Union to allow various different informations about called menu hook.
union PVR_MENUHOOK_DATA::data

Union to allow various different informations about called menu hook.

Class Members
int iEpgUid

If hook becomes called with type PVR_MENUHOOK_EPG becomes this used to identify source epg entry about them.

PVR_CHANNEL channel

If hook becomes called with PVR_MENUHOOK_CHANNEL becomes the channel from list passed to the add-on.

PVR_TIMER timer

On the opening of context menu from a timer list becomes it set by PVR_MENUHOOK_TIMER and the related timer information passed there.

PVR_RECORDING recording

With entry related to PVR_MENUHOOK_RECORDING or PVR_MENUHOOK_DELETED_RECORDING becomes this data used on menu hook call.

enum PVR_ERROR

Error return values for calls from Kodi to add-on who add-on inform about present error.

Enumerator
PVR_ERROR_NO_ERROR 

no error occurred

PVR_ERROR_UNKNOWN 

an unknown error occurred

PVR_ERROR_NOT_IMPLEMENTED 

the method that Kodi called is not implemented by the add-on

PVR_ERROR_SERVER_ERROR 

the backend reported an error, or the add-on isn't connected

PVR_ERROR_SERVER_TIMEOUT 

the command was sent to the backend, but the response timed out

PVR_ERROR_REJECTED 

the command was rejected by the backend

PVR_ERROR_ALREADY_PRESENT 

the requested item can not be added, because it's already present

PVR_ERROR_INVALID_PARAMETERS 

the parameters of the method that was called are invalid for this operation

PVR_ERROR_RECORDING_RUNNING 

a recording is running, so the timer can't be deleted without doing a forced delete
