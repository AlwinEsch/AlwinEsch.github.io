InfoTagRadioRDS
Python Development » Library - xbmc

Detailed Description

Kodi's radio RDS info tag class.

To get radio RDS info tag data of currently played PVR radio channel source.

Note
Info tag load is only be possible from present player class.
Also is all the data variable from radio channels and not known on begining of radio receiving.

Example:

1 ...
2 tag = xbmc.Player().getRadioRDSInfoTag()
3 
4 title = tag.getTitle()
5 artist = tag.getArtist()
6 ...
