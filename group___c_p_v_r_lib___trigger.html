CPVRLib_Trigger
PVRLIB

Detailed Description

Funtions to trigger PVR related updates on Kodi

Call of function in this class informs Kodi to do a update of related parts.

These are pure static functions them no other initialization need.

It has the header #include <kodi/pvr.api2/PVRLib.h> be included to enjoy it
