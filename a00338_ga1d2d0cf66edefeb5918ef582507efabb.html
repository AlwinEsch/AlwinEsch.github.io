Start or stop a server.

Parameters
[in]typUsed format of the returned language string
enum code: Description:
ADDON_ES_WEBSERVER To control Kodi's builtin webserver
ADDON_ES_AIRPLAYSERVER AirPlay is a proprietary protocol stack/suite developed by Apple Inc.
ADDON_ES_JSONRPCSERVER Control JSON-RPC HTTP/TCP socket-based interface
ADDON_ES_UPNPRENDERER UPnP client (aka UPnP renderer)
ADDON_ES_UPNPSERVER Control built-in UPnP A/V media server (UPnP-server)
ADDON_ES_EVENTSERVER Set eventServer part that accepts remote device input on all platforms
ADDON_ES_ZEROCONF Control Kodi's Avahi Zeroconf
[in]startstart (True) or stop (False) a server
[in]wait[opt] wait on stop before returning (not supported by all servers) (default is false)
Returns
true if successfull done
