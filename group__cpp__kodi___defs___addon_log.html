enum AddonLog

Detailed Description

Log file type definitions
These define the types of log entries given with kodi::Log() to Kodi.

Example:

#include <kodi/General.h>
kodi::Log(ADDON_LOG_ERROR, "%s: There is an error occurred!", __func__);
