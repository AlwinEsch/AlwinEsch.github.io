definition PVR_RECORDING_FLAG
C++ » Interface - kodi::addon » Addon type instances » PVR » Definitions, structures and enumerators » 5. Recording

Detailed Description

Bit field of independent flags associated with the EPG entry.
Values used by kodi::addon::PVRRecording::SetFlags().

Here's example about the use of this:

kodi::addon::PVRRecording tag;
tag.SetFlags(PVR_RECORDING_FLAG_IS_SERIES | PVR_RECORDING_FLAG_IS_PREMIERE);
