definition PVR_TIMER_TYPE
C++ » Interface - kodi::addon » Addon type instances » PVR » Definitions, structures and enumerators » 6. Timer

Detailed Description

PVR timer type attributes (kodi::addon::PVRTimerType::SetAttributes values).
To defines the attributes for a type. These values are bit fields that can be used together.

Example:

kodi::addon::PVRTimerType tag;
tag.SetAttributes(PVR_TIMER_TYPE_IS_MANUAL | PVR_TIMER_TYPE_IS_REPEATING);

#define PVR_TIMER_TYPE_REQUIRES_EPG_SERIES_ON_CREATE   0x00800000

This type should not appear on any create menus unless associated with an EPG tag with 'series' attributes.

Following conditions allow this:

  • kodi::addon::PVREPGTag::SetFlags have flag EPG_TAG_FLAG_IS_SERIES
  • kodi::addon::PVREPGTag::SetSeriesNumber > 0
  • kodi::addon::PVREPGTag::SetEpisodeNumber > 0
  • kodi::addon::PVREPGTag::SetEpisodePartNumber > 0

Implies PVR_TIMER_TYPE_REQUIRES_EPG_TAG_ON_CREATE.
