bool KodiAPI::AddOn::General::GetSettingBoolean ( const std::string &  settingName,
bool &  settingValue,
bool  global = false 
)

Get a boolean settings value for this add-on.

Parameters
[in]settingNameThe name of the setting to get.
[in]settingValueThe value.
[in]global[opt] If set to true becomes Kodi itself asked about a setting (default is false)
Returns
true if successfull done

Boolean type examples for add-on ./resources/settings.xml

  • bool
    • Shows a boolean check point to settings dialog
      1 <setting id="kodion.setup_wizard" type="bool" label="30025" default="true"/>
      2 <setting id="show-notifications" type="bool" label="32018" default="true" />
      • id="string" (required) - the name of the setting
      • label="id" (required) - an id from the language file that indicates which text to display.
      • default="true"|"false" (optional) - the default value.

Code example:

#include <kodi/api2/addon/General.hpp>
bool show_notifications = true;
if (!KodiAPI::AddOn::General::GetSettingBoolean("show-notifications", show_notifications))
KodiAPI::AddOn::General::Log(LOG_ERROR, "Couldn't get 'show-notifications' setting, falling back to default");
