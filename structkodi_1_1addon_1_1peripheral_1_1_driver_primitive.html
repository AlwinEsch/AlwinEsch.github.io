Detailed Description

DriverPrimitive

Base class for joystick driver primitives. A driver primitive can be:

1) a button 2) a hat direction 3) a semiaxis (either the positive or negative half of an axis) 4) a motor

The type determines the fields in use:

Button:

  • driver index

Hat direction:

  • driver index
  • hat direction

Semiaxis:

  • driver index
  • semiaxis direction

Motor:

  • driver index
