class Joystick

Detailed Description

Wrapper class providing additional joystick information
This is a child class to expand another class with necessary joystick data.

For data not provided by class Peripheral.

Used on:

  • kodi::addon::CInstancePeripheral::GetJoystickInfo()
  • kodi::addon::CInstancePeripheral::GetFeatures().
  • kodi::addon::CInstancePeripheral::MapFeatures().
  • kodi::addon::CInstancePeripheral::GetIgnoredPrimitives().
  • kodi::addon::CInstancePeripheral::SetIgnoredPrimitives().
  • kodi::addon::CInstancePeripheral::SaveButtonMap().
  • kodi::addon::CInstancePeripheral::RevertButtonMap().
  • kodi::addon::CInstancePeripheral::ResetButtonMap().
