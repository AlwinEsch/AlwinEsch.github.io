CAddonGUIControlTextBox
C++ Development » Library - GUILIB

Detailed Description

Used to show a multi-page piece of text

The text box control can be used to display descriptions, help texts or other larger texts. It corresponds to the representation which is also to be seen on the CAddonGUIDialogTextViewer.

It has the header #include <kodi/gui.api2/GUIControlTextBox.h> be included to enjoy it.

Note
The call of the control is only possible from the corresponding window as its class and identification number is required.
