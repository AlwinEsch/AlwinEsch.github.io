Detailed Description

Main PVR Add-on control class

The with #include <kodi/api3/pvr/Addon.hpp> included file brings support to create a pvr add-on on Kodi.

virtual PVR_ERROR GetAddonCapabilities ( PVR_ADDON_CAPABILITIES *  pCapabilities)
pure virtual

Get the list of features that this add-on provides.

Called by Kodi to query the add-on's capabilities.

Used to check which options should be presented in the UI, which methods to call, etc.

All capabilities that the add-on supports should be set to true. As default them set from Kodi to false.

Parameters
[out]pCapabilitiesThe add-on's capabilities, see PVR_ADDON_CAPABILITIES for detailed informations.
Returns
PVR_ERROR_NO_ERROR if the properties were fetched successfully. See PVR_ERROR for all possible values.
Warning
Valid implementation required.
