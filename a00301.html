Detailed Description

For control a radio button (as used for on/off settings).

ControlRadioButton(x, y, width, height, label[, focusOnTexture, noFocusOnTexture,
focusOffTexture, noFocusOffTexture, focusTexture, noFocusTexture,
textOffsetX, textOffsetY, alignment, font, textColor, disabledColor])

The radio button control is used for creating push button on/off settings in Kodi. You can choose the position, size, and look of the button. When the user clicks on the radio button, the state will change, toggling the extra textures (textureradioon and textureradiooff). Used for settings controls.

Note
This class include also all calls from Control
Parameters
[in]xinteger - x coordinate of control.
[in]yinteger - y coordinate of control.
[in]widthinteger - width of control.
[in]heightinteger - height of control.
[in]labelstring or unicode - text string.
[in]focusOnTexture[opt] string - filename for radio ON focused texture.
[in]noFocusOnTexture[opt] string - filename for radio ON not focused texture.
[in]focusOfTexture[opt] string - filename for radio OFF focused texture.
[in]noFocusOffTexture[opt] string - filename for radio OFF not focused texture.
[in]focusTexture[opt] string - filename for radio ON texture (deprecated, use focusOnTexture and noFocusOnTexture).
[in]noFocusTexture[opt] string - filename for radio OFF texture (deprecated, use focusOffTexture and noFocusOffTexture).
[in]textOffsetX[opt] integer - horizontal text offset
[in]textOffsetY[opt] integer - vertical text offset
[in]alignment[opt] integer - alignment of label
  • Flags for alignment used as bits to have several together:
    Defination name Bitflag Description
    XBFONT_LEFT 0x00000000 Align X left
    XBFONT_RIGHT 0x00000001 Align X right
    XBFONT_CENTER_X 0x00000002 Align X center
    XBFONT_CENTER_Y 0x00000004 Align Y center
    XBFONT_TRUNCATED 0x00000008 Truncated text
    XBFONT_JUSTIFIED 0x00000010 Justify text
[in]font[opt] string - font used for label text. (e.g. 'font13')
[in]textColor[opt] hexstring - color of enabled checkmark's label. (e.g. '0xFFFFFFFF')
[in]disabledColor[opt] hexstring - color of disabled checkmark's label. (e.g. '0xFFFF3300')
Note
You can use the above as keywords for arguments and skip certain optional arguments.
Once you use a keyword, all following arguments require the keyword.
After you create the control, you need to add it to the window with addControl().

Example:

2 self.radiobutton = xbmcgui.ControlRadioButton(100, 250, 200, 50, 'Enable', font='font14')
