Detailed Description

OK dialog

The functions listed below permit the call of a dialogue of information, a confirmation of the user by press from OK required.

These are pure static functions them no other initialization need.

It has the header #include <kodi/api2/gui/DialogOK.hpp> be included to enjoy it.
