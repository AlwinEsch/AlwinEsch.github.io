Live stream methods
C++ » 8. PVR » PVR Add-on (class KodiAPI::PVR::CAddonInterface)

Detailed Description

To read live stream from add-on if enabled by the properties

Used to open and close a stream to a channel, and optionally perform read operations on the stream

Note
All functions defined here are Calls from Kodi itself to the addon from class KodiAPI::PVR::CAddonInterface

Go back to normal functions from CAddonInterface
