definition EPG_TAG_FLAG
C++ » Interface - kodi::addon » Addon type instances » PVR » Definitions, structures and enumerators » 4. EPG Tag

Detailed Description

Bit field of independent flags associated with the EPG entry.
Values used by kodi::addon::PVREPGTag::SetFlags().

Here's example about the use of this:

kodi::addon::PVREPGTag tag;
tag.SetFlags(EPG_TAG_FLAG_IS_SERIES | EPG_TAG_FLAG_IS_NEW);
