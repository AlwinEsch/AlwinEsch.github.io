Channel group methods
C++ » 8. PVR » PVR Add-on (class KodiAPI::PVR::CAddonInterface)

Detailed Description

Call functions for Kodi to get channel group methods from add-on

Remarks
Only used by Kodi is PVR_ADDON_CAPABILITIES::bSupportsChannelGroups is set to true.
If a group or one of the group members changes after the initial import, or if a new one was added, then the add-on should call KodiAPI::PVR::Trigger::ChannelGroupsUpdate()
Note
All functions defined here are Calls from Kodi itself to the addon from class KodiAPI::PVR::CAddonInterface
