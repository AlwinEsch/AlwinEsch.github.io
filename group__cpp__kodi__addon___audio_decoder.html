Class: kodi::addon::CInstanceAudioDecoder

AudioDecoder add-on instance

A AudioDecoder (or screen saver) is a Kodi addon that blanks the screen or fills it with moving images or patterns when the computer is not in use. Initially designed to prevent phosphor burn-in on CRT and plasma computer monitors (hence the name), AudioDecoders are now used primarily for entertainment, security or to display system status information.

It has the header #include <kodi/AudioDecoder/AudioDecoder.h> be included to enjoy it.

This interface allow the creating of a AudioDecoders for Kodi, based upon DirectX or/and OpenGL rendering with C++ code.

The interface is small and easy usable, it have two function. One for the creation with Start() and for the render time with Render(), this both are required and must be implemented.

Further are several other functions available where the child class can ask about used device, display and few other parts.

Here as example what is minimum required to start on a AudioDecoder:

{
