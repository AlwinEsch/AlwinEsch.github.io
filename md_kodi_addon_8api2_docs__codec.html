#### *Group: Codec*

Functions

The with #include <kodi/addon.api2/Codec.h> given functions are required if demux becomes done on add-on itself and stream packet is requested from Kodi, e.g. on PVR DemuxPacket* DemuxRead(void);.

  • kodi_codec_t CAddOnLib_Codec::GetCodecByName(const std::string &strCodecName);
    • Get the codec id used by Kodi
      • strCodecName The name of the codec, normally the from ffmpeg used ones
      • return: The codec_id, or a codec_id with 0 values when not supported
  • DemuxPacket* CAddOnLib_Codec::AllocateDemuxPacket(int iDataSize);
    • Allocate a demux packet. Free with FreeDemuxPacket
      • iDataSize The size of the data that will go into the packet
      • return: The allocated packet
  • void CAddOnLib_Codec::FreeDemuxPacket(DemuxPacket* pPacket);
    • Free a packet that was allocated with AllocateDemuxPacket
      • pPacketThe packet to free

Classes

Class: CodecDescriptor::CodecDescriptor();

Adapter which converts codec names used by several backends into their FFmpeg equivalents.

  • CodecDescriptor::CodecDescriptor(kodi_codec_t codec, const char* name)
    • New class creation with given values, becomes done from static function GetCodecByName
      • codecThe from Kodi's add-on system used iden
      • nameString name used on add-on to translate for Kodi
        • Currently supported strings:
          • MPEG2AUDIO: For mpeg2 audio stream
          • MPEGTS: For mpeg2 video stream
          • TEXTSUB: For teletext subtitle
          • All others: Are equal to ffmpeg's identification strings
  • const std::string& CodecDescriptor::Name(void) const;
    • Returns the used codec string name
  • kodi_codec_t CodecDescriptor::Codec(void) const;
    • Returns the used Kodi's add-on codec structure.
  • static CodecDescriptor CodecDescriptor::GetCodecByName(const char* strCodecName);
    • Static creation function of class with string identification string, used to translate internal add-on format to on Kodi needed one.
      • strCodecNameThe used codec name, see above
      • return: The created class
