◆ makeLegalFilename()

makeLegalFilename (   ...)

Function: xbmcvfs.makeLegalFilename(filename)

Returns a legal filename or path as a string.

@param filename string - filename/path to make legal @return Legal filename or path as a string @note The returned value is platform-specific. This is due to the fact that the chars that need to be replaced to make a path legal depend on the underlying OS filesystem. This is useful, for example, if you want to create a file or folder based on data over which you have no control (e.g. an external API). \xrefitem python_v19 17. Example: @code{py} .. # windows >> xbmcvfs.makeLegalFilename('C://Trailers/Ice Age: The Meltdown.avi') C:\Trailers\Ice Age_ The Meltdown.avi # non-windows >> xbmcvfs.makeLegalFilename("///\\jk???lj????.mpg") /jk___lj____.mpg .. @endcode
