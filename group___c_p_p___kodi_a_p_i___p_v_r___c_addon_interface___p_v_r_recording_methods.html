Recording methods
C++ » 8. PVR » PVR Add-on (class KodiAPI::PVR::CAddonInterface)

Detailed Description

If add-on supports recordings read Kodi the available ones with them

Remarks
Only used by Kodi is PVR_ADDON_CAPABILITIES::bSupportsRecordings is set to true.
If a recording changes after the initial import, or if a new one was added, then the add-on should call KodiAPI::PVR::Trigger::RecordingUpdate()
Note
All functions defined here are Calls from Kodi itself to the addon from class KodiAPI::PVR::CAddonInterface
