Detailed Description

Add-on settings

Class: Settings()

This wrapper provides access to the settings specific to an add-on. It supports reading and writing specific setting values.

v20 Python API changes:
New class added.

Example:

...
settings = xbmc.Addon('id').getSettings()
...
