Keyboard Keyboard

Detailed Description

Keyboard class.

Keyboard([default, heading, hidden]) – Creates a new Keyboard object with default text heading and hidden input flag if supplied.

default : [opt] string - default text entry. heading : [opt] string - keyboard heading. hidden : [opt] boolean - True for hidden text entry.

example:

  • kb = xbmc.Keyboard('default', 'heading', True)
  • kb.setDefault('password') # optional
  • kb.setHeading('Enter password') # optional
  • kb.setHiddenInput(True) # optional
  • kb.doModal()
  • if (kb.isConfirmed()):
  • text = kb.getText()
