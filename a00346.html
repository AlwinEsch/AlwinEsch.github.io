CPlayerLib_PlayList
C++ Development » Library - PlayerLIB

Detailed Description

Playpack control list

Class to create a list of different sources to a continuous playback of this. This also supports a random mode (shuffle) to select the next source.

It has the header #include <kodi/player.api2/PlayList.h> be included to enjoy it.
