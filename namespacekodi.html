Detailed Description

PVR "C++" API interface.

In this field are the pure addon-side C++ data.

Note
Changes can be made without problems and have no influence on other PVR addons that have already been created.

Therefore, ADDON_INSTANCE_VERSION_PVR_MIN can be ignored for these fields and only the ADDON_INSTANCE_VERSION_PVR needs to be increased.
