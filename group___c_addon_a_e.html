Detailed Description

Audio Engine

Permits the use of the required functions of the add-on to Kodi. This class also contains some functions to the control.

These are pure static functions them no other initialization need.

It has the header #include <kodi/audioengine.api2/AudioEngineLib.h> be included to enjoy it.
