Detailed Description

Used to show a multi-page piece of text.

ControlTextBox(x, y, width, height[, font, textColor])

The text box is used for showing a large multipage piece of text in Kodi. You can choose the position, size, and look of the text.

Note
This class include also all calls from Control
Parameters
[in]xinteger - x coordinate of control.
[in]yinteger - y coordinate of control.
[in]widthinteger - width of control.
[in]heightinteger - height of control.
[in]font[opt] string - font used for text. (e.g. 'font13')
[in]textColor[opt] hexstring - color of textbox's text. (e.g. '0xFFFFFFFF')
Note
You can use the above as keywords for arguments and skip certain optional arguments.
Once you use a keyword, all following arguments require the keyword.
After you create the control, you need to add it to the window with addControl().

Example:

1 ...
2 # ControlTextBox(x, y, width, height[, font, textColor])
3 self.textbox = xbmcgui.ControlTextBox(100, 250, 300, 300, textColor='0xFFFFFFFF')
4 ...
