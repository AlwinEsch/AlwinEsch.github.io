void SetAttributes ( unsigned int  attributes)

required
Defines the attributes for this type (PVR_TIMER_TYPE_* constants).

To defines the attributes for a type. These values are bit fields that can be used together.

Example:

kodi::addon::PVRTimerType tag;
tag.SetAttributes(PVR_TIMER_TYPE_IS_MANUAL | PVR_TIMER_TYPE_IS_REPEATING);
