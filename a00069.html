Detailed Description

Used for a volume slider.

ControlSlider(x, y, width, height[, textureback, texture, texturefocus])

The slider control is used for things where a sliding bar best represents the operation at hand (such as a volume control or seek control). You can choose the position, size, and look of the slider control.

Note
This class include also all calls from Control
Parameters
[in]xinteger - x coordinate of control
[in]yinteger - y coordinate of control
[in]widthinteger - width of control
[in]heightinteger - height of control
[in]textureback[opt] string - image filename
[in]texture[opt] string - image filename
[in]texturefocus[opt] string - image filename
Note
You can use the above as keywords for arguments and skip certain optional arguments.
Once you use a keyword, all following arguments require the keyword.
After you create the control, you need to add it to the window with addControl().

Example:

1 ...
2 self.slider = xbmcgui.ControlSlider(100, 250, 350, 40)
3 ...
