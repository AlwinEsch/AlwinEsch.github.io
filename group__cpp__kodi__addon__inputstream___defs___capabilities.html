Detailed Description

Inputstream add-on capabilities
This class is needed to tell Kodi which options are supported on the addon.

If a capability is set to true, then the corresponding methods from kodi::addon::CInstanceInputStream need to be implemented.

Used on kodi::addon::CInstanceInputStream::GetCapabilities().
