Detailed Description

Callback functions for PVR add-on

Functions inside this class are used after an Kodi to PVR client request for transfer related data to Kodi.

This is a list of calls related to this class:

  • PVR_ERROR GetChannels(ADDON_HANDLE handle, bool bRadio);
  • PVR_ERROR GetChannelGroups(ADDON_HANDLE handle, bool bRadio);
  • PVR_ERROR GetChannelGroupMembers(ADDON_HANDLE handle, const PVR_CHANNEL_GROUP& group);
  • PVR_ERROR GetEPGForChannel(ADDON_HANDLE handle, const PVR_CHANNEL& channel, time_t iStart, time_t iEnd);
  • PVR_ERROR GetRecordings(ADDON_HANDLE handle, bool deleted);
  • PVR_ERROR GetTimers(ADDON_HANDLE handle);

These are pure static functions them no other initialization need.

It has the header #include <kodi/api2/pvr/Transfer.hpp> be included to enjoy it
