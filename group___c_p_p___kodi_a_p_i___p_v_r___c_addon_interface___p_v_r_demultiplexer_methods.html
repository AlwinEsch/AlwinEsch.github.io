Demultiplexer methods
C++ » 8. PVR » PVR Add-on (class KodiAPI::PVR::CAddonInterface)

Detailed Description

To do the demuxing of stream direct by the add-on if required from them

Used to open and close a stream to a recording, and perform read operations on the stream.

Remarks
Only used by Kodi is bHandlesDemuxing is set to true.
Note
All functions defined here are Calls from Kodi itself to the addon from class KodiAPI::PVR::CAddonInterface

virtual DemuxPacket* DemuxRead ( void  )
inlinevirtual

Read the next packet from the demultiplexer, if there is one.

Returns
The next packet.
If there is no next packet, then the add-on should return the packet created by calling AllocateDemuxPacket(0) on the callback. If the stream changed and Kodi's player needs to be reinitialised, then, the add-on should call AllocateDemuxPacket(0) on the callback, and set the streamid to DMX_SPECIALID_STREAMCHANGE and return the value.
The add-on should return nullptr if an error occured.
Remarks
Required if bHandlesDemuxing is set to true.
