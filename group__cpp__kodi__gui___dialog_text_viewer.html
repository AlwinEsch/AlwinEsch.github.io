Detailed Description

Namespace: kodi::gui::DialogTextViewer

Text viewer dialog

The text viewer dialog can be used to display descriptions, help texts or other larger texts.

In order to achieve a line break is a \n directly in the text or in the "./resources/language/resource.language.??_??/strings.po" to call with std::string kodi::general::GetLocalizedString(...);.

These are pure static functions them no other initialization need.

It has the header #include <kodi/gui/DialogTextViewer.h> be included to enjoy it.
