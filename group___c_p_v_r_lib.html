Detailed Description

Allow use of binary classes and function to use on add-on's

Permits the use of the required functions of the add-on to Kodi. This class also contains some functions to the control.

These are pure static functions them no other initialization need.

It has the header #include <kodi/pvr.api2/PVRLib.h> be included to enjoy it

static void AddMenuHook ( PVR_MENUHOOK *  hook)
static

Add or replace a menu hook for the context menu for this add-on.

Inserted hooks becomes called from Kodi on PVR system with PVR_ERROR CallMenuHook(const PVR_MENUHOOK& menuhook, const PVR_MENUHOOK_DATA &item);

Parameters
[in]hookThe hook to add

Here structure shows the necessary values the Kodi must be handed over around a menu by the PVR system to add.

/*
* Menu hooks that are available in the context menus while playing a
* stream via this add-on and in the Live TV settings dialog
*
* Only as example shown here! See always the original structure on related header.
*/
typedef struct PVR_MENUHOOK
{
unsigned int iHookId; /* This hook's identifier */
unsigned int iLocalizedStringId; /* The id of the label for this hook in g_localizeStrings */
PVR_MENUHOOK_CAT category; /* Category of menu hook */
} ATTRIBUTE_PACKED PVR_MENUHOOK;

Hook category Id's (PVR_MENUHOOK_CAT):

enum code: Id Description:
PVR_MENUHOOK_ALL 0 All categories
PVR_MENUHOOK_CHANNEL 1 For channels
PVR_MENUHOOK_TIMER 2 For timers
PVR_MENUHOOK_EPG 3 For EPG
PVR_MENUHOOK_RECORDING 4 For recordings
PVR_MENUHOOK_DELETED_RECORDING 5 For deleted recordings
PVR_MENUHOOK_SETTING 6 For settings

Code example:

#include <kodi/pvr.api2/PVRLib.h>
...
PVR_MENUHOOK hook;
hook.iHookId = 1;
hook.category = PVR_MENUHOOK_SETTING;
hook.iLocalizedStringId = 30107;
CPVRLib::AddMenuHook(&hook);
...

static bool RegisterMe ( void *  handle)
static

Resolve all callback methods.

Parameters
[in]handlePointer to the add-on
Returns
True when all methods were resolved, false otherwise.

Code example:

#include <kodi/addon.api2/AddonLib.h>
#include <kodi/pvr.api2/PVRLib.h>
// Insert one macro on your add-on for it
IMPLEMENT_KODI_ADDON_LIBRARY;
IMPLEMENT_KODI_PVR_LIBRARY;
...
ADDON_STATUS ADDON_Create(void* hdl, void* props)
{
...
// Enable access to all add-on to Kodi functions
if (!CAddonLib::RegisterMe(hdl))
return ADDON_STATUS_PERMANENT_FAILURE;
if (!CPVRLib::RegisterMe(hdl))
return ADDON_STATUS_PERMANENT_FAILURE;
...
}
Warning
To use the library must be one time the macro IMPLEMENT_KODI_PVR_LIBRARY and IMPLEMENT_KODI_ADDON_LIBRARY somewhere one time included.
