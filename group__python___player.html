Function Documentation

play (   ...)

Function: play([item, listitem, windowed, startpos])

Play a item.

Parameters
[in]item[opt] string - filename, url or playlist.
[in]listitem[opt] listitem - used with setInfo() to set different infolabels.
[in]windowed[opt] bool - true=play video windowed, false=play users preference.(default)
[in]startpos[opt] int - starting position when playing a playlist. Default = -1
Note
If item is not given then the Player will try to play the current item in the current playlist.

You can use the above as keywords for arguments and skip certain. optional arguments.
Once you use a keyword, all following arguments require the keyword.

Example:

1 ...
2 listitem = xbmcgui.ListItem('Ironman')
3 listitem.setInfo('video', {'Title': 'Ironman', 'Genre': 'Science Fiction'})
4 xbmc.Player().play(url, listitem, windowed)
5 xbmc.Player().play(playlist, listitem, windowed, startpos)
6 ...
