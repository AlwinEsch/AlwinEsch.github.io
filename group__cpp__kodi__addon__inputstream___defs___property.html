Detailed Description

Inputstream add-on property
This class is used to inform the addon of the desired stream.

Used on kodi::addon::CInstanceInputStream::Open().

The following table contains values that can be set with class InputstreamProperty :

Name Get call
The wanted stream URL GetURL
Special stream properties given by string id and string value GetProperties
The folder where addon library stored GetLibFolder
Folder where addon user data stored GetProfileFolder
